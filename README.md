# WeatherPro Dashboard

A modern, professional, and fully responsive weather dashboard web app providing real-time weather data, 3-day forecasts, hourly updates, air quality monitoring, astronomy info, marine weather, and severe weather alerts worldwide.

---

## üöÄ Features

```
-  Real-time weather data with location search by city or coordinates
-  3-day weather forecast with detailed temperatures, precipitation, humidity, wind, and UV index
-  24-hour hourly weather forecast with icons and temperature display
-  Air Quality Index (AQI) monitoring including CO, O3, NO2, SO2, PM2.5, PM10 pollutants
-  Astronomy data including sunrise, sunset, moonrise, moonset, moon phase, and illumination
-  Marine weather forecast with max/min temperature, max wind, and visibility
-  Severe weather alerts with headline and description
-  Location detection via Geolocation API or IP lookup as a fallback
-  Modern glassmorphism UI with dark theme and smooth animations
-  Fully accessible with WCAG 2.1 AA compliance, keyboard navigation, aria-labels, and focus indicators
-  SEO optimized with semantic HTML5, meta tags, Open Graph, Twitter Cards, structured data, robots.txt, and sitemap.xml
-  Progressive Web App (PWA) ready with manifest.json and service worker support
-  Cross-device support with responsive design for mobile, tablet, desktop, and large screens
-  Uses Google Fonts (Inter and Poppins), Font Awesome, Google Material Icons, and Lordicon animated icons
```
